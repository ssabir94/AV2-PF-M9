# PF-UF2-M9 | Mixtape Multimèdia

L’objectiu era integrar diferents continguts multimèdia dins d’una pàgina web i demostrar que entenem com funcionen i com s’utilitzen al web.

La pàgina recull diverses pràctiques que hem fet durant la unitat: comparació d’imatges, reproducció d’àudio, integració de vídeo i un formulari de contacte (junt amb les meves XXSS!)

---

# Estructura de la pàgina

La web està organitzada com una **Landing Page**, amb diferents seccions que es poden navegar des del menú superior.

Les seccions principals són:

- Imatges
- Àudio
- Vídeo
- Contacte
- Xarxes socials

Això permet que el visitant pugui recórrer la pàgina de manera senzilla i veure cada pràctica de forma clara.

També hi ha una **capçalera amb menú de navegació**, un **hero inicial** que introdueix el projecte i un **footer final**.

# Pràctica 1 – SVG vs Bitmap

En aquesta part vaig comparar **imatges vectorials (SVG)** amb **imatges de mapa de bits (PNG)**.

Per fer-ho, vaig crear tres dibuixos amb **SVG directament des del codi** i després els vaig comparar amb les seves versions en format raster.

La idea era mostrar que:

- les imatges **SVG es poden ampliar sense perdre qualitat**
- les imatges **PNG/JPG estan formades per píxels**

Per això les SVG es mostren grans al carrusel, per veure que mantenen la qualitat.  
En canvi les PNG, si es fan més grans o es fa zoom, es pot notar més fàcilment la pixelació.

També vaig afegir una **petita conclusió** sota els carrusels per resumir la diferència entre els dos formats.

---

# Pràctica 2 – Comparativa d’àudio

En aquesta secció es comparen diferents formats d’àudio utilitzant la mateixa cançó.

Els formats utilitzats són:

- **FLAC** → format sense pèrdua
- **MP3 128 kbps** → format comprimit més lleuger
- **MP3 320 kbps** → compressió amb més qualitat

L’objectiu és observar com canvien **la mida dels arxius i la qualitat del so** segons el tipus de compressió.

També vaig afegir una **taula comparativa** per resumir les característiques de cada format i fer més fàcil entendre les diferències.

---

# Pràctica 3 – Vídeo

En aquesta pràctica vaig crear un petit muntatge de vídeo utilitzant **Kdenlive**.

El vídeo es va editar amb:

- talls d’escenes
- música
- transicions
- alguns efectes

Després el vaig exportar en **format MP4** i integrar dins la pàgina web amb l’etiqueta `<video>`.

Al costat del reproductor hi ha un **petit resum tècnic** amb informació del vídeo (resolució, fps, format, etc.).

# Formulari de contacte

La pàgina inclou un **formulari de contacte** per simular l’enviament d’un missatge.

Aquest formulari utilitza **JavaScript** per validar que:

- tots els camps estiguin plens
- el correu tingui un format correcte

Quan hi ha algun error o quan el missatge s’envia correctament, apareix una notificació amb **SweetAlert**.
